Some of the most used HTML tags

  1. Anchor Tag(< a >):- It is used to create a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.

  2. Address Tag(< address >):- It is used to indicate that the enclosed content provides contact information for a person or people, or for an organization.

  3. Aside Tag(< aside >):- It is used to show content aside from the page content. This tag is used to describe the main object of the web page in a shorter way like a highlighter.

  4. Audio Tag(< audio >):- It is used to embed/add sound content like : songs , music etc.

  5. Bold Tag(< b >):- It is used to make the texts bold in HTML document.

  6. Body Tag(< body >):- The body tag in HTML is used to define the main content present inside an HTML page. In short it defines the document's body.

  7. Break Tag(< br >):- The break tag inserts a single line breaks in the document. This element has no end tag.

  8. Button Tag(< button >):- The button tag in HTML is used to define the clickable button. This tag is used to submit the content.

  9. Caption Tag(< caption >):- The caption tag is used to specify the caption of a table. Only one caption can be specified for one table.

  10. Delete Tag(< del >):- Delete tag is used to mark a portion of text which has been deleted from the document.

  11. Division Tag(< div >):- The div tag is used in HTML to make divisions of content in the web page (text, images, header, footer, navigation bar, etc).
    Simply it can be used for grouping of elements in HTML.

  12. Description List Tag(< dl >):- This tag in HTML is used to represent the description list.It have child elements,
    which are < dt >(description term/name) and < dd >(description details).

  13. Emphasis Tag(< em >):- It is used to italicize the texts in HTML document.

  14. Fieldset Tag(< fieldset >):- The fieldset tag in HTML5 is used to make a group of related elements in the form, and it creates the box over the elements

  15. Footer Tag(< footer >):- The footer tag in HTML is used to define a footer of HTML document. This section contains the footer information.

  16. Form tag(< form >):- This tag is used to create an HTML form for user input.
    This form is used basically for the registration process, logging into your profile on a website or creating your profile on a website, etc â€¦

  17. Heading Tags(< h1 >):- An HTML heading tag is used to define the headings of a page.
    These 6 heading elements are h1, h2, h3, h4, h5, and h6;
    with h1 being the highest level and h6 being the least.

  18. Head Tag(< head >):- The head tag in HTML is used to define the head portion of the document
    which contains metadata/information related to the document.

  19. Header Tag(< header >):- This tag defines a header for a document or section. The header tag is used to contain the information related to the title and heading of the related content.

  20. Horizontal Rule Tag(< hr >):- The hr tag in HTML stands for horizontal rule and is used to insert a horizontal rule.

  21. HTML Tag(< html >):- The html tag in HTML is used to define the root of HTML documents.

  22. Italic Tag(< i >):- It is also used to italicize the texts in HTML document. But using < em > instead of < i > is the best practice in html.

  23. Iframe Tag(< iframe >):- The iframe tag defines a rectangular region within the document in which the browser can display a separate document.

  24. Image Tag(< img >):- This tag is used to insert/add image files in an html document.

  25. Input Tag(< input >):- The input tag is used within < form > element to declare input controls that allow users to input data.

  26. Label Tag(< label >):- The label tag defines a label for an < input > element. This tag in HTML is used to provide a usability improvement for mouse users.

  27. Legend Tag(< legend >):- Defines a caption for a < fieldset > element. The legend tag is used to define the title for the child contents. The legend elements are the parent element.

  28. Link Tag(< link >):- Defines the relationship between a document and an external resource (most used to link to style sheets)

  29. Meta Tag(< meta >):- The meta tag is regularly used to give watchwords, portrayals, author data, and other metadata that might be utilized by the program to deliver the document accurately or in simple words, it provides important information about a document.

  30. Navigation Tag(< nav >):- The nav tag is used for declaring the navigational section in HTML documents. Websites typically have sections dedicated to navigational links, which enables users to navigate the site.

  31. Ordered List Tag(< ol >):- This tag is used to create an ordered list in html document. It has a child element which is < li > .

  32. Option Tag(< option >):- The option tag in HTML is used to choose an option from a Drop-Down menu/ list.

  33. Paragraph Tag(< p >):- This tag in HTML defines a paragraph. It have both opening and closing tags.

  34. Qoutation Tag(< q >):- This tag is used for short quotations. Like : quotation

  35. Strike Through Tag(< s >):- This tag is used to specify that the text content is no longer correct or accurate.
    This tag is similar but slightly different from < del > tag.

  36. Script Tag(< script >):- The script tag in HTML is used to define the client-side script.
    Under this tag JavaScript functions are written.

  37. Section Tag(< section >):- Section tag defines the section of documents such as chapters, headers, footers, or any other sections.

  38. Select Tag(< select >):- This tag is used to create a drop-down list in html.

  39. Small Tag(< small >):- The small tag in HTML is used to set small font sizes. It decreases the font size by one size

  40. Span Tag(< span >):- The HTML span element is a generic inline container for inline elements and content.

  41. Strong Tag(< strong >):- The strong tag in HTML is the parsed tag and is used to show the importance of the text. Make that text bold.

  42. Style Tag(< style >):- The style tag in HTML helps us to design the web page. Under this tag all CSS properties and values are written.

  43. Subscript Tag(< sub >):- This tag defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font.Subscript text can be used for chemical formulas, like H2O.

  44. Superscript Tag(< sup >):- This tag defines superscript text. Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font.Superscript text can be used for footnotes, like WWW[1] or any number with power , like 53.

  45. Table Tag(< table >):- This tag is used to create HTML Table which is an arrangement of data in rows and columns, or possibly in a more complex structure. It has many child elements which are :- < thead > , < tbody > , < tfoot > , < th > , < tr > , < td > etc.

  46. Textarea Tag(< textarea >):- This tag defines a multiline input control. Simply it creates a text input box which is generally used under < form > tag in HTML.

  47. Title Tag(< title >):- The title tag in HTML is used to define the title of HTML document. It sets the title in the browser toolbar.

  48. Underline Tag(< u >):- The underline tag in HTML stands for underline, and it is used to underline the text enclosed within the this tag.

  49. Unordered List Tag(< ul >):- This tag is used to create an unordered list in html document. It has a child element which is < li > .

  50. Video Tag(< video >):- This tag is used to add/embedded video content in an html document.
