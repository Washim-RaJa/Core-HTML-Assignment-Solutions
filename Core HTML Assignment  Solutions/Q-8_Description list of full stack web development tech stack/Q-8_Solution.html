Full Stack Web Development Tech Stack

Frontend Development

It is the visible part of website or web application which is responsible for user experience.
The user directly interacts with the front end portion of the web application or website.
The front end portion is built by using some languages which are HTML , CSS and Javascript.
Also some front end libraries and frameworks like:- AngularJS , React.js , Bootstrap , jQuery etc.

Backend Development

It refers to the server-side development of web application or website with a primary focus on how the website works.
It is responsible for managing the database through queries and APIs by client-side commands.
The back end portion is built by using some libraries, frameworks, and languages
which are:- PHP , C++ , Python , Java , Node.js etc.

Database

Database is the collection of inter-related data which helps in efficient retrieval, insertion and deletion of data from database
and organizes the data in the form of tables, views, schemas, reports etc.
Ex:- MongoDB , SQL etc.
