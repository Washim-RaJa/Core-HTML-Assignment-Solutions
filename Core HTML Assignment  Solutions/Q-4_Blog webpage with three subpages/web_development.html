Web Development

where we learn about creating, building, and maintaining of websites.

It consist of web design, web publishing, web programming, and database management.

It is a type application that runs on internet with help of a web browser.

In web development, web refers to websites, web pages or anythings that runs on internet.

Web Development can be classified as follows

  • Frontend
  • backend

Frontend

It is the visual & interactive part of a website. Where user can see the style and design of the website and interact with it.
In Frontend we learn about:-
  • HTML
  • CSS
  • JAVASCRIPT

Backend

It is the server side of a website. Where all the data and logic of the website are stored. This part of the website, user cannot see and interact with it directly.
In Backend we learn about:-
  • PHP
  • JAVASCRIPT
  • NODE.JS
  • MONGO.DB
  • PYTHON
