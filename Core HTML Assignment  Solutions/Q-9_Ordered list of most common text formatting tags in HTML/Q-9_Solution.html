Most common text formatting tags in HTML

  1. Bold Tag(< b > ):-
    • It is used to bold texts in a HTML document.
    • Using (< strong >) instead of (< b > ) is the best practice to bold text in HTML document.

  2. Strong Tag(< strong >):-
    • It is used to bold text and also to indicate that the particular text has importance.
    • Using this tag instead of (< b > ) is the best practice to bold text in HTML document.

  3. Italic Tag(< i > ):-
    • It is used to italicize text in a HTML document.
    • Using (< em >) instead of (< i > ) is the best practice to italicize text in HTML document.

  4. Emphasis Tag(< em >)
    • It is also used to italicize text in a HTML document but it emphasizes the important text or statement.
    • Using (< em >) instead of (< i > ) is the best practice to italicize text in HTML document.

  5. Mark Tag(< mark >)
    • It is used to highlight text in a HTML document.
    • We can also add style to this tag with CSS.

  6. Small Tag(< small >)
    • It is used to show text smaller in a HTML document.

  7. Delete Tag(< del >)
    • Upon using this tag, Browsers will usually strike a line through the deleted text.
    • This tag defines text that has been deleted from a document.

  8. Strike Through Tag(< s >)
    • Upon using this tag, Browsers will also strike a line through seleted text.
    • This tag represents contents that are no longer accurate or no longer relevant.
    • The (< s >) tag should not be used to define deleted text in a document, use the (< del >) tag for that.

  9. Insert Tag(< ins >)
    • This tag defines a text that has been inserted into a document.
    • Browsers will usually underline the inserted text.

  10. Subscript Tag(< sub >)
    • This tag defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font.
    • Subscript text can be used for chemical formulas, like H2O

  11. Superscript Tag(< sup >)
    • This tag defines superscript text. Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font.
    • Superscript text can be used for footnotes, like WWW[1] or any number with power , like 53
